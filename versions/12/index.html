Connect to Informatica System

Establish trusted connection between D360 and Informatica tenants

Configure details for the Informatica tenant, so that a trusted connection between them can be provided. The important items are "User-Name" and "Password"

Documentation
Documentation

Validation Criteria

Ensure that the connection is successfully established and credentials are validated.

Choose Business Entity

Select the primary business entity from Informatica

Picker populated with a list of Informatica's primary business entities: Customer, Organization, Product, Supplier

Validation Criteria

Ensure that a valid business entity is selected from the list.

Review DLOs and Field Types

Review Data Load Objects and their field types

Review the Data Load Objects (DLOs) and verify that all field types are correctly mapped.

Documentation
Documentation

Validation Criteria

All DLOs and field types should be reviewed and verified.

Review and Extend Mappings

Review existing mappings and extend as needed

Review the existing field mappings and extend them to include any additional required mappings.

Documentation
Documentation

Validation Criteria

All required mappings should be reviewed and extended as necessary.
